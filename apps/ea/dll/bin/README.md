# PTWebSocket DLL Binaries

Pre-built binaries from CI. Do not edit - these are generated by GitHub Actions.

## Files

| File | Platform | Config | Size | EA Target |
|------|----------|--------|------|-----------|
| `PTWebSocket.dll` | x64 | Release | ~200 KB | MT5 `MQL5/Libraries/` |
| `PTWebSocket32.dll` | Win32 | Release | ~160 KB | MT4 `MQL4/Libraries/` |
| `Debug/PTWebSocket.dll` | x64 | Debug | ~1.8 MB | MT5 (debug builds) |
| `Debug/PTWebSocket32.dll` | Win32 | Debug | ~1.4 MB | MT4 (debug builds) |

## Rebuilding

Source code is in `../PTWebSocket/`. CI workflow: `.github/workflows/build-dll.yml`.

To rebuild locally (requires MSVC on Windows):
```bash
mkdir build && cd build
cmake -G "Visual Studio 17 2022" -A x64 ../PTWebSocket
cmake --build . --config Release --parallel
```
